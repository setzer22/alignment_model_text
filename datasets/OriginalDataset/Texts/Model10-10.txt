The fault detector reports a failure to the user or user has a suspicion of their own fault. 
The user shall examine the failure. 
The user rejects the failure of the fault detector or the user confirms the failure of the fault detector. 
If the user confirms the failure of the fault detector, he informs the customer and the manager. 
The user fixes the fault at the measuring device. 
The user shares the results of the repairs carried out with the fault detector. 
The user will inform the customer about the resolution of the interference. 
The user will inform the manager about the resolution of the interference.
